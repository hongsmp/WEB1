Javascript

Javascript, often abbreviated as JS, is a high-level, interpreted programming language. It is a language which is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm. Alongside HTML and CSS, JS is one of the 3 core techologies of the World Wide Web. JS enables interactive web pages and thus is a essential part of web applications. The vast majority of website use it and all major web browsers have a dedicated JavaScript engine to execute it
